# 🌐 Landing Page - Tânia Orfei

Este projeto foi desenvolvido como atividade da disciplina de Desenvolvimento Front-End do curso de Análise e Desenvolvimento de Sistemas.

O objetivo foi criar uma Landing Page pessoal utilizando apenas **HTML5** e **CSS3**, aplicando conceitos de estrutura semântica, estilização e responsividade.

## ✨ Funcionalidades

- Apresentação pessoal
- Seção "Sobre Mim"
- Lista de habilidades
- Projetos (exemplos fictícios)
- Curiosidades
- Links para redes profissionais
- Layout responsivo para desktop e dispositivos móveis

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
